# Landing_page_SkillCraft
Overview

This project includes a simple, fixed navigation menu that:


Stays visible at the top of every page
Changes style (shadow + slightly smaller padding) when the page is scrolled
Highlights links on hover with a color change and underline


Files

File Purpose:- 
task.html -Page markup 
task..css - Styling for the layout
and 
task.js - Scroll behavior for the navigation menu
